### 🎨 设计理念

本图标库主要针对 `Sub-Store` 与 `Quantumult X` 进行设计。采用纯色设计，简约而不简单，旨在为用户提供一种清晰、美观、一致的视觉体验。


### 🔧 使用方法

1. 打开 `Sub-Store` 或 `Quantumult X` 的设置页面。
2. 导航到图标设置部分。
3. 选择想要替换的图标，并提供本图标库中相应图标的链接。
